Gahara Naufal Rayhan

Perkenalkan nama saya Gahara Naufal Rayhan.Saya biasa di panggil oleh teman saya Garra ataupun Gaga. Saya lahir di Bandung tanggal 1- Agustus 2002. Saya adalah anak pertama dari dua bersaudara. Saya tinggal di jalan Raden Koyong No 38 Tanah Baru Bogor

Setiap orang pasti memiliki hobi, dan hobi saya adalah bermain game. Saya memiliki hobi bermain game karena ayah saya dahulu suka bermain game juga, jadi sejak kecil saya sudah tau berbagai macam game. Dari hobi saya ini saya pernah mengikuti pertandingan game besar seperti contohnya IGC 2016. Dalam IGC 2016 saya mengikuti pertandingan Clash Royale dan sayangnya saya kalah pada saat 8 besar dibawah ini adalah foto saya ketika di igc 2018.

Dan dari hobi saya itu saya memiliki seorang idola yaitu caster terbaik di Indonesia yang bernama Ericko Lim. Ericko lim merupakan youtuber yang sekarang sedang berkembang dengan konten game. Selain menjadi caster dan youtuber, ericko lim merupakan salah satu pro player LOL (League Of Legend), dan ketika ia menjadi pro player di game LOL, ia pernah menjadi perwakilan Indonesia dalam ASIAN GAME. Walaupun dia kasar dalam berbicara akan tetapi sesekali dia menginspirasi.
